# Snow Battle
Snow battle é um jogo multiplayer do tipo arena shooter 4v4

## Conexão Multiplayer
Para conectar é necessário que ambos os dispositivos estejam conectados na mesma rede de internet. 
O host irá abrir um servidor e os outros jogadores devem inserir o ip do host (aparece na tela dele) para conectar

## Tutorial
- movimentação: setas direcionais ou wasd
- disparo: clique esquerdo do mouse ou espaço
